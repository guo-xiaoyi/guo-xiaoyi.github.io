Evolution of Reference Points over Time

with Matthias Weber, Enrico de Giorgi, and Andrija Lukovic

The formation and evolution of reference points in a dynamic setting are rare topics in discussions of reference-dependent preferences. Based on existing theories about the formation of reference points, we conduct a controlled economic experiment on a representative sample. Our aim is to establish a unified framework describing reference points in dynamic settings.

Evaluating Home Effect for Professional Cycling

This research is the first in the field trying to comprehensively understand the home effect in professional cycling, based on Tour de France. We use modified causal forests to estimate the home effects of French riders in different stages of Tour de France. The result is that there is a considerable negative home effect for home riders in home stage, and such effect exhibits heterogeneous variations.

coolTree: Enhanced Experimental Interface with More Templates

This is a side project I did to enhance the experimental interface of oTree, which is a Python package for designing and conducting economic experiments. The new version includes more templates for different types of experiments, making it easier for researchers to set up their studies without extensive coding.

This project is available on GitHub and is still under development. Stay tuned for updates! :)
