Current Projects

Evolution of Reference Points over Time

with Matthias Weber, Enrico de Giorgi, and Andrija Lukovic

Reference-dependent preferences rarely consider how reference points evolve in dynamic settings. We run a controlled experiment on a representative sample to build a unified framework that captures how reference points form and adjust over time.

Correlation Neglect in Intertemporal Decisions

with Matthias Weber

We study how decision makers handle correlation in intertemporal choice. The project is in early stages and more details will follow soon.

Evaluating Home Effect for Professional Cycling

Using Tour de France data, we estimate home effects for French riders with modified causal forests. The analysis uncovers a negative home effect with meaningful heterogeneity across stages.

coolTree: Experimental Interface Extensions

A lightweight revamp of the oTree interface with additional templates for economic experiments. The project is open-source and under active development.
